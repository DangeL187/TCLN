-----------------------------------------------------
Total Control over the Local Network (TCLN) by DangeL
-----------------------------------------------------
Folders:
bin - all executable applications
build - build files
doc - documentation for every TCLN's module
lib - libraries that you will need to build TCLN
src - TCLN's source code
test - TCLN's files used for testing
versions - TCLN's previous versions
-----------------------------------------------------
Source files:
TCLN.cpp - main application
-----------------------------------------------------
Compilation:
Go to the bin folder and type:
./build.sh
or
./build_and_run.sh
